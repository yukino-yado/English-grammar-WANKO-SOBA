# 英文法わんこそば 本実装版 v3.10

# 英文法わんこそば v3.3 仕様メモ

## v3.3追加仕様

1. 教師用アプリに「問題を編集する」を追加する。
2. クリックすると `/question-editor.html` を別窓で開く。
3. 問題編集画面では、既存問題を含めて全単元の日本語文と英文のみ編集できる。
4. 編集は上書き扱い。
5. 穴埋め文、正解、選択肢、解説、学年、単元、難易度は編集しない。
6. 編集済み問題・追加問題は教師側のシステム上で強調し、自動生成時の参考データとして扱う。
7. 生徒側には編集済み表示を出さない。
8. ページ遷移は `/teacher.html`、`/question-editor.html`、`/index.html` の絶対パスを使う。

## v3.3 共有保存仕様

- 教師用編集データは `localStorage` だけでなく、Vercel Blobへ保存する。
- 共有APIは `/api/teacher-package`。
- `GET /api/teacher-package`：共有教師データを読み込む。
- `POST /api/teacher-package`：共有教師データを上書き保存する。
- 教師用画面、問題編集画面、生徒用画面は起動時に共有データを読み込む。
- 教師用で保存した内容は、端末をまたいで共有される。
- Blob未設定時はエラー表示を出し、ブラウザ内保存は維持する。
- 共有データの内部形式は従来の教師用パッケージと同じ。


# 英文法わんこそば v2.10 仕様メモ

## アプリ分離

- 生徒用アプリ：`index.html`
- 教師用アプリ：`teacher.html`

生徒用の画面構成は従来のまま維持し、教師用は別画面として追加する。

## 連携方式

### 静的版の連携

- 教師用データ保存キー：`wankoSobaTeacherPackage:v1`
- 生徒用進捗保存キー：`wankoSobaGrammarProgress:v3`
- 教師用で「生徒版へ反映」を押すと、教師用パッケージをLocalStorageへ保存する
- 生徒用は起動時に教師用パッケージを読み込み、以下を反映する
  - 追加問題：`teacherBank`
  - お品書き上書き：`MENU_EXPLANATIONS`

### 別端末対応

LocalStorageは端末・ブラウザごとの保存であるため、別端末では教師用 teacher.html で配布用データを取り込んでから、生徒用 index.html を開く。生徒用 index.html からは追加問題を直接編集・取り込みできない。

### 本格運用で必要になるもの

- 教師ログイン
- 生徒ログインまたはクラスコード
- 問題データベース
- お品書き説明データベース
- 校舎・学年・クラス別配信
- 生徒の正答率集計

## 配布用データの内部形式（開発者向け）

```json
{
  "version": 1,
  "updatedAt": "2026-06-04T00:00:00.000Z",
  "menuOverrides": {
    "一般動詞の文②": {
      "normal": "説明文",
      "examples": ["Does she listen to music?"],
      "secret": "発展説明"
    }
  },
  "questions": [
    {
      "grade": 1,
      "unit": "一般動詞の文②",
      "level": "medium",
      "jp": "私の兄は毎日音楽を聴きますか。",
      "fullSentence": "Does my brother listen to music every day?",
      "blankSentence": "___ my brother ___ to music every day?",
      "blankAnswer": "Does listen",
      "choices": ["Does", "Do", "listen", "listens"],
      "explanation": "主語が my brother なので Does を使い、動詞は原形 listen にします。"
    }
  ]
}
```

## 教師用機能

- お品書き編集
  - 通常説明
  - 例文
  - 裏・お品書き説明
- 追加問題作成
  - 学年
  - 単元
  - 難易度
  - 日本語
  - 全文英文
  - 穴埋め英文
  - 穴埋め答え
  - 選択肢
  - 解説
- 類題候補提示
  - テキスト貼り付け
  - `.txt` / `.json` 読み込み
  - 単元に応じたテンプレート類題を提示
  - 候補を編集して追加
- 配布用データの作成／取り込み

## 生徒用への影響

- 生徒用画面の見た目は大きく変えない
- 文法説明は「お品書き」、単元選択は「本日の単元表」として分ける
- 各学年の単元表には「全部選ぶ」「全部外す」ボタンを置く
- 「先生からの一杯」は既存の空きスペースに置く
- 教師用で追加された問題は、生徒用では「先生の仕込み棚」として単元別に選択可能。先生の仕込み棚にも「全部選ぶ」「全部外す」を置く
- 生徒用では追加問題の直接編集・直接取り込みはできない
- 教師用で編集されたお品書き説明は、生徒用のお品書きに反映される

## v2.10 修正

- えらんで一杯・うめて一杯で、主語などの弱い空欄ではなく、その単元の文法の核になる部分を空欄にする方針を強化しました。

### v3.2 仕様追記

- `/question-editor.html` の検索条件に難易度を追加。
- 検索条件は「難易度・学年・単元・検索（任意）」とし、検索ボタン押下後に該当問題を全件表示する。
- 検索語は日本語文・英文を対象にする。
- 難易度が `all` の基準問題は、各難易度で検索した場合にも表示対象とする。

## v3.4 仕様追加

### ホーム表示時の問題事前生成

生徒用アプリのホーム画面が開かれた時点で、各学年・単元・難易度ごとに生成候補を作成し、進捗データ内の `preGeneratedBank` に保存する。

### 教師用エディターでの検索対象

「問題を編集する」ページでは、以下を検索・編集対象に含める。

- 基準問題
- 追加問題
- 生徒が解いた履歴に残っている問題
- 保存英文に残っている問題
- 復習待ちに残っている問題
- ホーム画面表示時に事前生成された問題

編集可能項目は引き続き「日本語文」と「英文」のみとする。
