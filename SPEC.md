# 英文法わんこそば v2.8 仕様メモ

## アプリ分離

- 生徒用アプリ：`index.html`
- 教師用アプリ：`teacher.html`

生徒用の画面構成は従来のまま維持し、教師用は別画面として追加する。

## 連携方式

### 静的版の連携

- 教師用データ保存キー：`wankoSobaTeacherPackage:v1`
- 生徒用進捗保存キー：`wankoSobaGrammarProgress:v3`
- 教師用で「生徒版へ反映」を押すと、教師用パッケージをLocalStorageへ保存する
- 生徒用は起動時に教師用パッケージを読み込み、以下を反映する
  - 追加問題：`teacherBank`
  - お品書き上書き：`MENU_EXPLANATIONS`

### 別端末対応

LocalStorageは端末・ブラウザごとの保存であるため、別端末では教師用 teacher.html で配布用データを取り込んでから、生徒用 index.html を開く。生徒用 index.html からは追加問題を直接編集・取り込みできない。

### 本格運用で必要になるもの

- 教師ログイン
- 生徒ログインまたはクラスコード
- 問題データベース
- お品書き説明データベース
- 校舎・学年・クラス別配信
- 生徒の正答率集計

## 配布用データの内部形式（開発者向け）

```json
{
  "version": 1,
  "updatedAt": "2026-06-04T00:00:00.000Z",
  "menuOverrides": {
    "一般動詞の文②": {
      "normal": "説明文",

      "secret": "発展説明"
    }
  },
  "questions": [
    {
      "grade": 1,
      "unit": "一般動詞の文②",
      "level": "medium",
      "jp": "私の兄は毎日音楽を聴きますか。",
      "fullSentence": "Does my brother listen to music every day?",

      "explanation": "主語が my brother なので Does を使い、動詞は原形 listen にします。"
    }
  ]

```

## 教師用機能

- お品書き編集
  - 通常説明
  - 例文
  - 裏・お品書き説明
- 追加問題作成
  - 学年
  - 単元
  - 難易度
  - 日本語
  - 全文英文
  - 穴埋め英文
  - 穴埋め答え
  - 選択肢
  - 解説
- 類題候補提示
  - テキスト貼り付け
  - `.txt` / `.json` 読み込み
  - 単元に応じたテンプレート類題を提示
  - 候補を編集して追加
- 配布用データの作成／取り込み

## 生徒用への影響

- 生徒用画面の見た目は大きく変えない
- 文法説明は「お品書き」、単元選択は「本日の単元表」として分ける
- 各学年の単元表には「全部選ぶ」「全部外す」ボタンを置く
- 「先生からの一杯」は既存の空きスペースに置く
- 教師用で追加された問題は、生徒用では「先生の仕込み棚」として単元別に選択可能。先生の仕込み棚にも「全部選ぶ」「全部外す」を置く
- 生徒用では追加問題の直接編集・直接取り込みはできない
- 教師用で編集されたお品書き説明は、生徒用のお品書きに反映される


## v2.8 更新

- 各学年・各単元の問題プールを大幅に増やしました。
- 単元ごとに、その学年で扱う文法だけを使うように問題プールを整理しました。
- 「thatを使う文・間接疑問文」では that節の問題が確実に出るよう、that節の候補を増やしました。
- 同じ問題ばかり出る状況を避けるため、ランダム生成前の候補文を増やしました。

- すべての基本単元で最低30種類以上の出題候補を用意し、単元選択時でも同じ問題だけになりにくくしました。
- 保存キーは v5 に変更しています。


## v2.8 修正

- えらんで一杯・うめて一杯で、主語などの弱い空欄ではなく、その単元の文法の核になる部分を空欄にする方針を強化しました。
- that節は「that」だけでなく「glad that」「think that」「sure that」など、文法として覚えたいまとまりを問う形に調整しました。
- 疑問詞の問題は「What」だけでなく「What do」「Where does」「How can」など、語順の核まで問う形に調整しました。
- 存在構文は「There is / There are」のまとまりを問う形に調整しました。


## v2.8 追加
- 教師用の「生徒用を確認」から開くと、特盛り・裏お品書きなどを開放状態で確認できます。称号は解放しません。
- 裏・お品書きの学年縛りを外し、通常生徒側は回数条件のみで開放します。
- 教師用で単元ごとに裏・お品書きを作る／作らないを選べます。


## v2.8 画面整理

- 先生からの一杯は生徒用トップに維持。
