# 英文法わんこそば 本実装版 v2.10

HTML / CSS / JavaScript だけで動くブラウザアプリです。

## 起動方法

ZIPを解凍して、以下のファイルをブラウザで開きます。

- 生徒用：`index.html`
- 教師用：`teacher.html`

## 生徒用でできること

- お客様の人数＝学年選択
- お品書き／裏・お品書き（文法説明）
- 本日の単元表
- 各学年の単元を全部選ぶ／全部外す
- 小盛り〜特盛り
- 10〜30杯選択
- 制限時間 5〜30秒
- えらんで一杯
- ならべて一杯
- うめて一杯
- 書ききり一杯
- ランダム問題生成
- 書ききり一杯は過去に解いた問題から出題
- 解説表示
- 履歴保存
- おかわり復習
- 苦手単元チェック
- 円グラフによる理解度表示
- 称号
- EX開放条件
- アニメーション演出
- 先生からの一杯
- メニュー項目をグループ化して見やすく整理

## 教師用でできること

- お品書き説明の編集
- 裏・お品書き説明の編集
- 例文の編集
- 先生がやらせたい問題の手入力追加
- ファイル・テキストから類題候補の提示
- 類題候補を編集して追加
- 追加問題の一覧確認・削除
- 教師用データの配布用データ作成／取り込み
- 生徒版への反映

## 教師用から生徒用への反映

同じフォルダ内で `teacher.html` と `index.html` を開く構成です。

### 同じブラウザで確認する場合

1. `teacher.html` を開く
2. お品書きや問題を追加する
3. 「生徒版へ反映」を押す
4. `index.html` を開く、または開き直す
5. 生徒用の「先生からの一杯」に追加問題が反映される

### 別端末・別ブラウザで使う場合

1. 教師用で「配布用データを作る」
2. 生徒端末側で teacher.html を開き、配布用データを取り込む
3. 同じ端末で index.html を開く
4. 生徒用に追加問題が反映される

※この版はサーバーやデータベースを使わない静的アプリです。全生徒端末へ自動配信する本格運用には、Firebase / Supabase などのデータベース連携が必要です。

## ファイル構成

- `index.html`：生徒用アプリ
- `script.js`：生徒用ロジック
- `style.css`：共通・生徒用スタイル
- `teacher.html`：教師用アプリ
- `teacher.js`：教師用ロジック
- `teacher.css`：教師用スタイル
- `README.md`：使い方
- `SPEC.md`：仕様メモ

## 注意

教師用の「類題候補」は、ローカルのテンプレートと簡易解析で提示します。AIによる高精度な類題生成を行う場合は、サーバー側でAPI連携する構成が必要です。


## v2.10 更新

- 各学年・各単元の問題プールを大幅に増やしました。
- 単元ごとに、その学年で扱う文法だけを使うように問題プールを整理しました。
- 「thatを使う文・間接疑問文」では that節の問題が確実に出るよう、that節の候補を増やしました。
- 同じ問題ばかり出る状況を避けるため、ランダム生成前の候補文を増やしました。

- すべての基本単元で最低30種類以上の出題候補を用意し、単元選択時でも同じ問題だけになりにくくしました。
- 保存キーは v5 に変更しています。


## v2.10 修正

- えらんで一杯・うめて一杯で、主語などの弱い空欄ではなく、その単元の文法の核になる部分を空欄にする方針を強化しました。
- that節は「that」だけでなく「glad that」「think that」「sure that」など、文法として覚えたいまとまりを問う形に調整しました。
- 疑問詞の問題は「What」だけでなく「What do」「Where does」「How can」など、語順の核まで問う形に調整しました。
- 存在構文は「There is / There are」のまとまりを問う形に調整しました。


## v2.10 追加

- 教師用で単元ごとに裏・お品書きを作る／作らないを選べます。
